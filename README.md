# AI Recommendation Project

## Project Description
AI Movie Recommendation System is a Python-based application that recommends movies based on user interests and preferred genres. It analyzes user preferences, matches them with available movie genres, and provides personalized movie suggestions. The system also stores recommendation history for future reference.

## Features
- Personalized movie recommendations
- Genre-based movie matching
- User preference analysis
- Recommendation score calculation
- Recommendation history saving
- Simple command-line interface

## Technologies Used
- Python
- JSON
- Data Structures
- Recommendation Algorithms

## How to Run
1. Clone the repository
2. Open terminal
3. Run:
   python main.py
